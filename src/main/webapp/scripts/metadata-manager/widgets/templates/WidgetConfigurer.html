Call Function via Widget Binding

                    
Widget Bindings, which simply bind event listeners to other widgets, can be used to call a function on a widget when the bound widget dispatches the specified event (and optionally the data/payload included with the event matches a query).

                    
 
"widgetBindings" : {
  "nameOfWidgetWeAreBindingTo" : [{
     "event" : "change",
     "action" : "setVisible",
     "args" : [ "${nameOfWidgetWeAreBindingTo.isVisible}" ]
   }]
}
